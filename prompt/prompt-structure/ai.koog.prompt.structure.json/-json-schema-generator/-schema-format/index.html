Simple format, embed all type definitions. Won't work for more complicated cases, e.g. where subclass refers back again to the base class

JsonSchema

This is a very simplified version, not fully according to the JSON schema specification (https://json-schema.org). The full specification is quite complex, and we don't need to support this complexity here.
